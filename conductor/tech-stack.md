# Tech Stack: Ekline Website

## Core Frontend
- **Framework:** [React 19](https://react.dev/) - Utilizing the latest features for a performant, component-based UI.
- **Build Tool:** [Vite 7](https://vite.dev/) - Providing a fast development environment and optimized production builds.
- **Language:** JavaScript (ESM) - Standard modern JavaScript for application logic.

## UI & Styling
- **Styling:** CSS3 - Standard CSS for layout and design (e.g., `App.css`, `index.css`).
- **Animations:** [Framer Motion](https://www.framer.com/motion/) - Powers smooth, declarative transitions and interactive UI elements.
- **Icons:** [Lucide React](https://lucide.dev/) - A clean, consistent icon library for modern interfaces.

## Development & Tooling
- **Linting:** [ESLint 9](https://eslint.org/) - Ensures code quality and adherence to established JavaScript standards.
- **Package Management:** npm (via `package-lock.json`) - Managing project dependencies and scripts.
